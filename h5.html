Tehtävä a, CRUD sovellus Django kehityspalvelimelle

Asennetaan Django kehitysympäristö

Asennetaan virtualenv, jonka mukana tulee uusimmat python paketit, kuten Django

                        (env) sanna@sanna-virtualbox:~$ sudo apt-get -y install virtualenv

Luodaan /env hakemisto, jolla on käytössää uusimmat paketit ja python3

                        (env) sanna@sanna-virtualbox:~$ virtualenv --system-site-packages -p python3 env/

Käytetään uutta virtual environmentia

                        (env) sanna@sanna-virtualbox:~$ source env/bin/activate

Tarkastetaan, että asennus menee virtuaaliseen ympäristöön

                        (env) sanna@sanna-virtualbox:~$ which pip
                        /home/sanna/env/bin/pip

asennetaan micro tekstieditori-ohjelma

                        (env) sanna@sanna-virtualbox:~$ sudo apt-get install micro

Luodaan requirements.txt tiedosto ja kirjoitetaan sinne ajettava käsky 'django' 

                        (env) sanna@sanna-virtualbox:~$ micro requirements.txt # kirjoitettu sisällöksi django

Tarkastetaan sisältö

                        (env) sanna@sanna-virtualbox:~$ cat requirements.txt 
                        django

Asennetaan django
